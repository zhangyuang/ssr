# 官方文档

官方文档请查看 [http://doc.ssr-fc.com/](http://doc.ssr-fc.com/)

## getting start

```bash
$ npm start # 本地开发模式运行，单进程 支持 前端 HMR 前端静态资源走本地 webpack 服务
$ npm run prod # 模拟生产环境运行，多进程，前端资源走静态目录
$ npm run stop # 生产环境停止服务
```
